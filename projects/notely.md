# Project: Notely

## Description
Notely is a collaborative music review application that allows users to create groups, add albums, and share reviews with group members. The application features user authentication, group management, album tracking, and a review system with voting capabilities. It's built with a React frontend and Node.js/Express backend using MongoDB for data storage.

## Goals
- Create a platform for music enthusiasts to collaboratively review albums
- Enable users to form groups with friends to share music opinions
- Provide a clean, intuitive interface for tracking album listening progress
- Implement a fair review system that reveals opinions simultaneously
- Support deployment on Vercel for easy access

## Notes
- The application uses MongoDB Atlas for cloud database storage
- JWT authentication is implemented for user sessions
- MusicBrainz API integration provides album metadata and cover art
- React Query handles data fetching and state management
- Tailwind CSS and shadcn/ui provide the UI components
